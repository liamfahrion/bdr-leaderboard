# Profile Photos

Drop your photo files into this folder. Each file must be named to match the rep's name in the `PHOTOS` config object in `index.html` (case-insensitive).

## Expected filenames

| Rep name in sheet | File to drop here |
|---|---|
| Dalynn | `dalynn.jpg` |
| Mike | `mike.jpg` |
| Danicka | `danicka.jpg` |
| Jackson | `jackson.jpg` |
| Ringo | `ringo.jpg` |
| Delaney | `delaney.jpg` |
| Charles | `charles.jpg` |
| Jake | `jake.jpg` |
| Wyatt | `wyatt.jpg` |

## Notes

- **Format:** `.jpg` or `.png` both work. Update the path in `PHOTOS` if you use `.png`.
- **Size:** Square crops work best. Anything from 200×200 px up is fine — the board scales them to a circle.
- **Missing photos:** If a file is absent or fails to load, the board automatically shows a circular initials avatar instead. No photo is required for the board to run.
- **Adding a new rep:** Add an entry to the `PHOTOS` object in `index.html` and drop the matching file here.
